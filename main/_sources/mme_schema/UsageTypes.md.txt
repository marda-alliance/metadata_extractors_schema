# Enum: UsageTypes




_This enumeration specifies the execution environment for the `Extractor`._



URI: [UsageTypes](UsageTypes)

## Permissible Values

| Value | Description | Comments |
| --- | --- | --- |
| cli | Access the `Extractor` as a command line executable. The executable as well as<br/>the invocation syntax, parametrised using arguments, shall be supplied in the<br/>`command` slot. |  |
| python | Access the `Extractor` as a Python function. The function should be available<br/>when `import <setup>` is called, and should be specified including relevant<br/>namespaces. The function signature, parametrised using arguments, shall be<br/>supplied in the `command` slot. |  |




## Slots

| Name | Description |
| ---  | --- |
| [method](method.md) | Usage invocation method, e |
